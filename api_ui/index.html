File Operations

POST /upload-file Upload a new file to a folder ‚ñº
URL
/api/upload-file
Headers
authorization: YOUR_PASS_CODE Content-Type: multipart/form-data
Form Data
file: [Select image/video file] (required) folderName: "e-commerce/products" (optional - if not provided, uploads to root)
cURL
curl -X POST /api/upload-file \ -H "authorization: YOUR_PASS_CODE" \ -F "file=@/path/to/image.jpg" \ -F "folderName=e-commerce/products"
PUT /update-file Update/Replace an existing file ‚ñº
URL
/api/update-file
Headers
authorization: YOUR_PASS_CODE Content-Type: multipart/form-data
Form Data
file: [Select new image/video file] (required) publicId: "e-commerce/products/abc123" (required)
cURL
curl -X PUT /api/update-file \ -H "authorization: YOUR_PASS_CODE" \ -F "file=@/path/to/new-image.jpg" \ -F "publicId=e-commerce/products/abc123"
GET /get-file Get file details by publicId ‚ñº
URL
/api/get-file
Headers
authorization: YOUR_PASS_CODE
Query Parameters
publicId: "e-commerce/products/abc123" (required) resourceType: "image" or "video" (optional, default: "image")
cURL
curl -X GET "/api/get-file?publicId=e-commerce/products/abc123&resourceType=image" \ -H "authorization: YOUR_PASS_CODE"
DELETE /delete-file Delete a file by publicId ‚ñº
URL
/api/delete-file
Headers
authorization: YOUR_PASS_CODE Content-Type: application/json
Body (JSON)
{ "publicId": "e-commerce/products/abc123", "resourceType": "image" }
cURL
curl -X DELETE /api/delete-file \ -H "authorization: YOUR_PASS_CODE" \ -H "Content-Type: application/json" \ -d '{"publicId": "e-commerce/products/abc123", "resourceType": "image"}'
